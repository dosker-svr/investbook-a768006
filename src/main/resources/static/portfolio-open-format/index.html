Бэкап и восстановление данных

Бэкап данных может пригодиться, если вы хотите защитить данные от сбоев вашего компьютера.

На этой странице вы можете:

  1. Выполнить бэкап и восстановление данных.
  2. Импортировать в Investbook портфель из другого приложения учета инвестиций, если оно поддерживает "Portfolio Open Format" (за инструкцией по получению файла в указанном формате обратитесь к документации или разработчикам этих приложений).
  3. Выполнить экспорт данных в формате "Portfolio Open Format" для переноса портфеля в другое приложение учета инвестиций.

Вернуться на главную страницу



1. Для того чтобы получить файл для бэкапа или экспорта в другое приложение, нажмите на ссылку экспортировать данные в "Portfolio Open Format".

2. Для того чтобы восстановить данные, загрузите ранее полученный по предыдущему пункту файл бэкапа.

В бэкап не включаются курсы валют ЦБ РФ и значения индекса S&P 500. После восстановления из бэкапа вы, возможно, захотите выгрузить эти данные через страницу Форм.

Если счет уже существует в БД, то отсутствующие сделки будут добавлены из бэкапа в существующий счет, дублирование сделок не произойдет. При желании перед восстановлением можно удалить имеющиеся счета.
